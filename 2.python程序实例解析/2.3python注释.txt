Python中单行注释以 # 开头，实例如下：
# print("123")

多行注释可以用多个 # 号，还有 ''' 和 """：
#!/usr/bin/python3
 
# 第一个注释
# 第二个注释
 
'''
第三注释
第四注释
'''
 
"""
第五注释
第六注释
"""
print ("Hello, Python!")
执行以上代码，输出结果为：

Hello, Python!

注释快捷键
windows下visual studio code：选中多行 ctrl+/ 可以多行注释或者反注释

（1）vi/vim如何添加或删除多行注释方法
在使用vi/vim编辑器对代码或者配置文件编辑的时候，经常会需要对多行添加注释或者删除多行的注释。
 
进入vi/vim编辑器，按CTRL+V进入可视化模式（VISUAL BLOCK）

移动光标上移或者下移，选中多行的开头

选择完毕后，按大写的的I键，此时下方会提示进入“insert”模式，输入你要插入的注释符，例如#，

最后按ESC键，你就会发现多行代码已经被注释了

删除多行注释的方法，同样 Ctrl+v 进入列选择模式，移到光标把注释符选中，按下d，注释就被删除了。


（2）使用替换命令

:% s/^/#/g 来在全部内容的行首添加 # 号注释
:1,10 s/^/#/g 在1~10 行首添加 # 号注释

撤销注释：
:%s/^#//g
