1.3. 小总结（注释的概念和作用）

  • 注释：在程序代码中对程序代码进行解释说明的文字。

  • 作用：注释不是程序，不能被执行，只是对程序代码进行解释说明，让别人可以看懂程序代码的作用，能够大大增强程序的可读性。

2. 注释的分类

2.1. 单行注释

以#开头，#右边的所有文字当作说明，而不是真正要执行的程序，起辅助说明作用


# 我是注释，可以在里写一些功能说明之类的哦
print('hello world')
