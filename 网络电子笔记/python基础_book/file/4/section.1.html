运行后的现象:

想一想：

如果一个程序在不同的地方需要输出“佛祖镇楼”，程序应该怎样设计？

    if 条件1:
        输出‘佛祖镇楼’

    ...(省略)...

    if 条件2:
        输出‘佛祖镇楼’

    ...(省略)...

如果需要输出多次，是否意味着要编写这块代码多次呢？

小总结:
  • 函数就是一个具有独立功能的代码块，可以提高代码的复用性。
